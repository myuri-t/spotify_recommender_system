# Spotify Recommender System

This project builds a music recommendation system using Spotify audio feature data and machine learning techniques.

## Overview
The goal of this project is to recommend songs based on similarity between audio features such as tempo, danceability, energy, and acousticness.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing and cleaning
- Feature analysis of Spotify music attributes
- Similarity-based recommendation model
- Music recommendation output based on feature similarity
